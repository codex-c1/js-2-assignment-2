Game of Telephone

Each "person" gets assigned information one-at-a-time. If someone changes their story, no one else will know about it until they get re-assigned a value.

To change this, go to 'script.js', and change the values for the following variables:

    aliceSays
    bobSays
    carolSays
    danSays
    ellieSays
    frankSays
    gregSays
    harrySays
    ivySays
    jackSays
